Manual WhatsApp QR Generator

Paste the QR URL from the GitHub Actions logs:

How to use:

  1. Run the GitHub Actions workflow
  2. Look for "QR Code URL" in the logs
  3. Copy the entire URL (starts with 2@...)
  4. Paste it in the box above
  5. Click "Generate QR Code"
  6. Scan with WhatsApp → Settings → Linked Devices → Link a Device

Troubleshooting:

  • Make sure you copy the ENTIRE URL
  • The URL should start with "2@"
  • Scan within 2 minutes of generation
  • If it fails, run the workflow again for a fresh QR
